# Phone Book Manager

## 📞 About the Project

The Phone Book Manager is a beginner-friendly Python application that allows users to add, view, and search contacts. The application uses file handling to store contact information permanently in a text file, helping users understand data persistence and basic file operations.

## ✨ Features

- Add new contacts
- View all saved contacts
- Search for a contact by name
- Store contacts permanently using a text file
- Simple menu-driven interface

## 🛠️ Technologies Used

- Python
- File Handling

## 🎯 Learning Outcomes

This project helped me learn:

- Reading and writing text files
- Python functions
- File handling
- Search operations
- Menu-driven programming

## 🚀 Future Improvements

- Update contact details
